# SNS投稿ライブラリ

Obsidian VaultのSNS投稿候補を美しく整理し、いつでも閲覧・検索できるモバイル最適化Webサイトです。

## 📱 特徴

- **モバイルファースト設計**: スマホでの閲覧に最適化
- **ダークモード**: 目に優しい洗練されたデザイン
- **高速検索**: キーワードで即座に投稿を検索
- **カテゴリーフィルター**: AI、ビジネス、学習で絞り込み
- **ワンタップコピー**: 投稿文を簡単にコピー
- **150件の投稿**: すべての投稿候補を一元管理

## 🚀 使い方

### サイトを開く

1. スマホのブラウザで以下のURLを開く:
   ```
   file:///c:/Users/masan/Desktop/Masa HP/sns-posts/index.html
   ```

2. または、PCで開いてスマホでアクセスできるようにする

### 投稿を探す

1. **検索**: 上部の検索バーにキーワードを入力
2. **フィルター**: カテゴリーチップをタップして絞り込み
3. **閲覧**: 投稿カードをタップして詳細を表示

### 投稿をコピー

1. 投稿カードをタップしてモーダルを開く
2. 「投稿文をコピー」ボタンをタップ
3. SNSアプリに貼り付けて使用

## 🔄 データ更新

Markdownファイルを更新した後、以下のコマンドでデータを再生成:

```bash
cd "c:/Users/masan/Desktop/Masa HP/sns-posts"
node extract-data.js
```

## 📁 ファイル構成

```
sns-posts/
├── index.html          # メインHTML
├── style.css           # スタイルシート
├── app.js              # アプリケーションロジック
├── data.js             # 投稿データ（自動生成）
├── extract-data.js     # データ抽出スクリプト
└── README.md           # このファイル
```

## 🎨 カスタマイズ

### カラーテーマの変更

`style.css`の`:root`セクションで色を変更できます:

```css
:root {
    --primary: #2563eb;      /* メインカラー */
    --secondary: #8b5cf6;    /* セカンダリカラー */
    --accent: #f59e0b;       /* アクセントカラー */
    /* ... */
}
```

### カテゴリーの追加

1. `extract-data.js`の`markdownFiles`に新しいカテゴリーを追加
2. `categoryInfo`に表示情報を追加
3. `node extract-data.js`を実行

## 📊 統計

- **総投稿数**: 150件
- **カテゴリー**: 3つ（AI、ビジネス、学習）
- **各カテゴリー**: 50件ずつ
